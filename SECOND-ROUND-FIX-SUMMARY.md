#### 🟡 **問題4：adv-9 第2步時間表述混淆**

**狀態**：⚠️ → ✅ 已修復

```javascript
// 修改前
options: ['周一上午12點', '周一中午12點', '周一下午12點', '當天午夜12點']
// ❌ '上午12點'、'中午12點'、'下午12點' 在中文中表述混淆

// 修改後
options: ['周一凌晨12點', '周一中午12點', '周一下午11點', '周日午夜12點']
// ✅ 正確的時間表述：
//    - 凌晨12點 = 00:00
//    - 中午12點 = 12:00
//    - 下午11點 = 23:00
//    - 午夜12點 = 00:00
```

**時間表述問題**：
```
❌ 錯誤用法：
  - "上午12點"（上午應該是1-11點）
  - "下午12點"（12點本身是中午）

✅ 正確用法：
  - "凌晨12點" = 00:00
  - "中午12點" = 12:00
  - "午夜12點" = 00:00（午夜 = 凌晨）
```
